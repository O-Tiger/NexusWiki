# Módulos

NexusSlime está organizado en **13 módulos de funcionalidades**, cada uno gestionando un sistema de juego distinto. Todos los módulos están incluidos en el JAR final y habilitados por defecto.

---

| Módulo | Descripción |
| --- | --- |
| [Core](core.md) | Ítems personalizados, sistema PDC, registro de ítems, mochilas, waypoints |
| [Esenciales](essentials.md) | Homes, warps, TPA, AFK, cárcel, +40 comandos QoL |
| [Economía](economy.md) | Dinero, créditos, /sell, /worth, /baltop |
| [Clanes](clans.md) | Creación de clan, territorio, mejoras, cofre de clan, chat de clan |
| [Seguridad](security.md) | Auth BCrypt, anti-bot CAPTCHA, anti-lag, anti-dupe |
| [Discord](discord.md) | Bot JDA, vinculación de cuenta, sincronización de roles, avisos en vivo |
| [Crystal Defense](crystal-defense.md) | Minijuego cooperativo de defensa de Cristales Ender por oleadas |
| [Votifier](votifier.md) | Servidor Votifier V1/V2 independiente, rachas de votos, clasificación |
| [Sueños](dreams.md) | Sistema de cutscene de sueño con variantes de sueño y pesadilla |
| [Protecciones](protections.md) | Reclamación de regiones, banderas de protección, sistema de duelo |
| [Mobs Personalizados](custom-mobs.md) | Jefes definidos en YAML con tablas de botín, formas de IA y huevos de spawn |
| [Crafteo Infinito](infinity-crafting.md) | Estaciones de crafteo multibloques con formato de receta YAML |
| [Integración Twitch](twitch.md) | Vinculación de cuenta, alertas en vivo, relay de chat, sorteos para espectadores |
